Bible Audio JSON API

A static JSON API for the King James Version (KJV) Bible Audio Narration.

API Endpoints

GET /api/v1/all.json

Get the complete Bible audio data.

GET /api/v1/ot.json

Get Old Testament data only.

GET /api/v1/nt.json

Get New Testament data only.
